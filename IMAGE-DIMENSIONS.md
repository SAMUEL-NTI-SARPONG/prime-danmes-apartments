# Image & Logo Dimension Guide — Prime Danmes Apartments

All recommended dimensions ensure crisp display on retina/HiDPI screens (2×).

---

## 1. Logo / Site Icon

- **Displayed at**: 40 × 40 px (navbar & footer icon area)
- **Recommended file size**: **80 × 80 px** (2× retina) — square
- **Format**: PNG with transparent background, or SVG
- **Where used**: Navbar, footer, admin sidebar, browser tab (favicon)

> For the favicon (`public/favicon.ico`), also provide a **32 × 32 px** ICO or PNG.

---

## 2. Hero / Banner Image (Homepage)

- **Displayed as**: Full-width background, cropped via `object-cover`
- **Recommended file size**: **1920 × 1080 px** (16:9 landscape)
- **Minimum**: 1600 × 900 px
- **Format**: JPEG (quality 80–85) or WebP
- **Where used**: Homepage hero section

---

## 3. Apartment Card Thumbnail

- **Aspect ratio**: **4:3** (`aspect-4/3`)
- **Displayed at**: ~400 px wide (card width)
- **Recommended file size**: **800 × 600 px**
- **Format**: JPEG or WebP
- **Where used**: Apartment listing cards, admin listing cards

---

## 4. Apartment Detail Gallery (Main Image)

- **Aspect ratio**: **16:9** on mobile (`aspect-video`), **21:9** on desktop (`aspect-[21/9]`)
- **Recommended file size**: **1680 × 720 px** (21:9 covers both ratios)
- **Minimum**: 1200 × 680 px
- **Format**: JPEG or WebP
- **Where used**: Apartment detail page main gallery image

---

## 5. Apartment Detail Gallery Thumbnails

- **Displayed at**: 80 × 56 px (mobile) / 112 × 72 px (desktop)
- **Recommended file size**: Same image as the gallery main — thumbnails are auto-scaled
- **No separate images needed**

---

## 6. Admin Apartment Form — Image Previews

- **Aspect ratio**: **16:9** (`aspect-video`)
- **Displayed at**: ~100 px wide thumbnails
- **Recommended file size**: Same images uploaded for the gallery
- **No separate images needed**

---

## 7. About Page Hero Image

- **Displayed as**: Full-width, height 256 px (mobile) to 448 px (desktop)
- **Recommended file size**: **1600 × 900 px** (16:9)
- **Format**: JPEG or WebP
- **Where used**: About page top hero

---

## 8. About Page Section Image

- **Displayed at**: ~50% width on desktop, full width on mobile; height 288–384 px
- **Recommended file size**: **800 × 600 px** (4:3)
- **Format**: JPEG or WebP
- **Where used**: "Proudly Ghanaian Hospitality" section

---

## 9. Uploaded Images (via Admin Panel)

- **Max file size**: 5 MB per image
- **Accepted formats**: JPEG, PNG, WebP, AVIF
- **Recommended dimensions**: **1200 × 800 px** minimum for gallery quality
- **Best practice**: Upload the largest reasonable size; the app auto-scales for display

---
